# Recicle Mais
“Recicle Mais” é um projeto voltado para a conscientização sobre a reciclagem. Com a construção de um site de caráter informativo onde tudo, desde o design até os ajustes finais, foi formulado para ser limpo e acessível.
A paleta de cores em diferentes tons de verde faz referência ao tema da reciclagem, além de formas arredondadas que trazem suavidade para o usuário. Além disso, conta com animações de “flip card” dinâmicas e modernas.
Sobre a programação, o site é responsivo, ou seja, é adaptável a diversos dispositivos, como computadores, tablets e celulares.

### Tecnologias utilizadas:
- HTML;
- CSS;
- JavaScript;
- Site Figma para design.
